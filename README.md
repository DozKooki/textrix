Fun text effects for Python: typing animation, rainbow colors, glitch effect, and more.

## Installation
```bash
pip install textrix
````

## Usage

```python
import textrix

textrix.typing("Hello Dishan!", delay=0.1)
textrix.rainbow("Rainbow text")
textrix.glitch("Glitch effect!", duration=2)
```
